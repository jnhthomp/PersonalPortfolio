AlphaBlog

This is a blog site that I created using Ruby on Rails. You can view browse articles, categories, and users while anonymous. Upon acount creation you can create your own articles to display on the site. This project taught me about the MVC model in Ruby on Rails, hashing user passwords, authenticating users, and creating/managing a relational database.

Food Order Application

This is a react project that allows the user to create and submit an order to a fictional restaurant. It uses Firebase API to both fetch the menu data as well as submit user orders. This application taught me about using fetch requests within React to handle incoming and outgoing data. I also learned how to utilize React context in order to manage sitewide state data.

Stalks

A Ruby on Rails application which allows you to make a profile and add stock tickers to follow to your portfolio. You can also add other users to see what stocks they are interested in and tracking. This application taught me about using third party gems such as Devise for simpler user authentification and API usage in Ruby on Rails applications.

My Skills

I use an assortment of different technologies and skills to create and manage applications for your business. Below are the ones I use most often but I am always looking for new things to learn and new challenges to undertake.

React.js

I utilize the React framework to create modern, modular, and responsive user interfaces. Including managing complex data and states with built in features such as React Context or third party libraries like Redux

Ruby on Rails

Utilizing the MVC framework Ruby on Rails is a powerful backend that can be used to manage data between the server and users for large applications. In the past I have used this to manage user authorization and real time updates within the app with websocket

PostgreSql

Using SQL I am able to find and extract data from company databases to help business leaders make data-driven decisions on how to best support their customers.

Git

Git is my version control of choice. This allows me to work with other developers on your projects as well as ensure that if anything were to go wrong we can roll back to a previous working version.

Workflow Management

Using workflow applications we can automate processes within your business. In the past I have used this to drastically reduce the workload for customer facing employees so they have more time to focus on each individual customer.

Project Management

I integrate user feedback and current processes to determine the best way to create new processes. This allows the user to quickly get up to speed with new features with as little friction as possible.
